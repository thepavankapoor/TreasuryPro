# TreasuryPro
Dashboard for quick stock details like financial statements, ratios, interest rates, etc. It uses data from Yahoo Finance, and WTO.


How to run:

1. Open the repository folder in the command prompt
2. type "python app.py" and press enter
3. http://127.0.0.1:5000 - paste this in your URL bar and load the dashboard.
4. Type any US stock name in the search bar and hit go.
